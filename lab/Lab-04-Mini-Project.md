# Lab 04 — AI Vision Mini Project
### สร้างโปรเจกต์ AI ของตัวเอง

---

## วัตถุประสงค์ / Objectives

หลังจาก Lab นี้นักเรียนจะสามารถ:

- เลือกโจทย์ปัญหาที่สนใจและออกแบบ AI solution ได้
- เขียน prompt ที่เหมาะสมกับ use case ของตัวเองได้
- นำเสนอผลงานให้ผู้อื่นเข้าใจได้

---

## เลือกโปรเจกต์ / Choose Your Project

เลือก 1 โปรเจกต์จากตัวเลือกด้านล่าง หรือคิดขึ้นมาเอง:

| # | โปรเจกต์ | คำอธิบาย |
|---|---------|----------|
| 1 | **Waste Classification** | จำแนกขยะเป็นประเภทรีไซเคิล/อินทรีย์/ทั่วไป |
| 2 | **Plant Health Analysis** | วิเคราะห์สุขภาพต้นไม้จากสีใบและลักษณะ |
| 3 | **Object Counting** | นับจำนวนวัตถุในพื้นที่ที่กำหนด |
| 4 | **Classroom Monitoring** | ตรวจสอบสภาพห้องเรียน (มีคน/ว่าง/สกปรก) |
| 5 | **Desk Cleanliness Inspection** | ประเมินความเป็นระเบียบของโต๊ะทำงาน |
| 6 | **ไอเดียของตัวเอง** | คิดขึ้นมาเอง — ขอคำแนะนำจากอาจารย์ |

---

## ขั้นตอน / Steps

### ขั้นที่ 1 — กำหนดโจทย์ปัญหา (5 นาที)

ตอบคำถามเหล่านี้:

1. **ปัญหาคืออะไร?** เช่น "โรงเรียนมีปัญหาการแยกขยะไม่ถูกต้อง"
2. **AI จะช่วยได้อย่างไร?** เช่น "กล้องวิเคราะห์และแจ้งเตือนประเภทขยะที่ถูกต้อง"
3. **ผลลัพธ์ที่ต้องการรับจาก AI คืออะไร?** เช่น "ชื่อประเภทขยะ + วิธีทิ้ง"

---

### ขั้นที่ 2 — เขียน Prompt (10 นาที)

เริ่มจาก prompt ใน `prompts.py` แล้วปรับให้เหมาะกับโปรเจกต์ของคุณ

**Framework สำหรับเขียน prompt ที่ดี:**

```
1. บอก Context: "คุณคือระบบ AI ที่ช่วย[สิ่งที่ต้องการ]"
2. อธิบาย Task: "วิเคราะห์ภาพและ[สิ่งที่ต้องทำ]"
3. กำหนด Format: "ตอบในรูปแบบ: [ประเภท]: [รายละเอียด]"
4. ระบุภาษา: "ตอบเป็นภาษาไทย"
```

**ตัวอย่าง prompt สำหรับ Waste Classification:**

```
คุณคือระบบ AI สำหรับจำแนกขยะในโรงเรียน

วิเคราะห์ภาพขยะในภาพนี้และบอก:
1. ประเภทขยะ: รีไซเคิล / อินทรีย์ / ทั่วไป / อันตราย
2. วัสดุที่เห็น: (เช่น ขวดพลาสติก, เปลือกผลไม้)
3. วิธีทิ้งที่ถูกต้อง: (1-2 ประโยค)

ตอบสั้นๆ ชัดเจน ใช้ภาษาที่นักเรียนเข้าใจง่าย
```

---

### ขั้นที่ 3 — ทดสอบและปรับปรุง (15 นาที)

1. เปิด Web UI `http://localhost:8000`
2. เลือก preset ที่ใกล้เคียง หรือวาง prompt ของตัวเองในช่อง Prompt
3. ชี้กล้องไปที่วัตถุที่เกี่ยวกับโปรเจกต์ของคุณ
4. คลิก "Capture & Analyze"
5. ดูผล แล้วถามตัวเองว่า:
   - ผลถูกต้องไหม?
   - จะปรับ prompt อย่างไรให้ดีขึ้น?
6. ทดสอบซ้ำจนพอใจ (อย่างน้อย 3 รอบ)

---

### ขั้นที่ 4 — เตรียมนำเสนอ (10 นาที)

เตรียมตอบคำถามเหล่านี้เมื่อนำเสนอ:

1. **โปรเจกต์ของฉันทำอะไร?** (1-2 ประโยค)
2. **Prompt ที่ใช้คืออะไร?** (แสดงให้ดู)
3. **ผลการทดสอบเป็นอย่างไร?** (แสดง screenshot หรือ demo สด)
4. **ถ้าพัฒนาต่อ จะเพิ่มอะไร?** (ไอเดีย 1-2 อย่าง)

## สิ่งที่ได้เรียนรู้ / What You Learned

- การสร้าง AI solution เริ่มจากการกำหนดปัญหาให้ชัดเจน

- Vision AI สามารถนำไปประยุกต์ใช้กับปัญหาจริงในชีวิตประจำวันได้
