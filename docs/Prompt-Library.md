# Prompt Library — ESP32-CAM AI Vision Workshop
### คลัง Prompt สำหรับการวิเคราะห์ภาพด้วย AI

คู่มือนี้รวบรวม prompt ที่ทดสอบแล้วว่าได้ผลดีกับ Gemini Vision และ Claude Vision
แบ่งเป็นหมวดหมู่ตามการใช้งาน พร้อม template ที่นักเรียนสามารถนำไปปรับใช้ได้

---

## หลักการเขียน Prompt ที่ดี / Prompt Engineering Basics

```
[บทบาท] + [Task] + [Format ที่ต้องการ] + [ภาษา]
```

**ตัวอย่าง:**
```
คุณคือผู้เชี่ยวชาญด้านสิ่งแวดล้อม [บทบาท]
วิเคราะห์ขยะในภาพและจำแนกประเภท [Task]
ตอบเป็น bullet point ภาษาไทย [Format + ภาษา]
```

## 1. Mini-Project Presets (5 หัวข้อหลัก)

### 1.1 Waste Classification — จำแนกประเภทขยะ

**Basic:**
```
วิเคราะห์และจำแนกขยะในภาพ ระบุประเภท (รีไซเคิล/อินทรีย์/ทั่วไป/อันตราย) และวิธีทิ้งที่ถูกต้อง
```

**Detailed:**
```
คุณคือระบบ AI จำแนกขยะอัจฉริยะ วิเคราะห์ภาพและตอบในรูปแบบนี้:
🗑️ ประเภทขยะ: [ชื่อ]
♻️ หมวดหมู่: รีไซเคิล / อินทรีย์ / ทั่วไป / อันตราย
✅ วิธีทิ้งที่ถูกต้อง: [คำอธิบาย 1-2 ประโยค]
⚠️ หมายเหตุ: [ถ้ามี]
```

**JSON Format (สำหรับนักเรียนที่สนใจโปรแกรมมิ่ง):**
```
Analyze the waste in this image and respond in JSON:
{"type": "...", "category": "recyclable|organic|general|hazardous", "disposal": "...", "confidence": "high|medium|low"}
```

### 1.2 Plant Health Analysis — วิเคราะห์สุขภาพต้นไม้

**Basic:**
```
วิเคราะห์สุขภาพต้นไม้ในภาพ บอกระดับสุขภาพ (ดี/ปานกลาง/ต้องการดูแล) และสาเหตุ
```

**Detailed:**
```
คุณคือนักพฤกษศาสตร์ผู้เชี่ยวชาญ วิเคราะห์ต้นไม้ในภาพและตอบ:
🌿 ชนิดต้นไม้: [ถ้าระบุได้]
💚 ระดับสุขภาพ: ดีมาก / ดี / ปานกลาง / ต้องการดูแลด่วน
🔍 สิ่งที่สังเกตเห็น: [สีใบ, จุด, ใบเหี่ยว, ฯลฯ]
💧 การรดน้ำ: ควรรดเพิ่ม / เพียงพอแล้ว / รดมากเกินไป
🌱 คำแนะนำ: [2-3 ข้อ]
```

---

### 1.3 Object Counting — นับจำนวนวัตถุ

**Basic:**
```
นับจำนวนวัตถุทุกชิ้นในภาพ แยกประเภท และบอกจำนวนรวม
```

**Detailed:**
```
นับและจำแนกวัตถุทุกชิ้นในภาพนี้อย่างละเอียด ตอบในรูปแบบ:
[ชื่อวัตถุ]: [จำนวน]
...
รวมทั้งหมด: [จำนวน] ชิ้น
หมายเหตุ: [วัตถุที่นับได้ยากหรือมองไม่ชัด]
```

**Inventory Mode:**
```
You are an inventory system. Count all items visible in this image.
Format: {"items": [{"name": "...", "count": 0, "condition": "good|damaged|unclear"}], "total": 0}
```

### 1.4 Classroom Monitoring — ตรวจสอบห้องเรียน

**Basic:**
```
วิเคราะห์สภาพห้องเรียนในภาพ: จำนวนคน, ความเป็นระเบียบ, สภาพแสงสว่าง
```

**Detailed:**
```
ประเมินสภาพห้องเรียนนี้:
👥 จำนวนคน: [ตัวเลข หรือ ไม่มี]
📚 ความเป็นระเบียบ: ดีมาก / ดี / พอใช้ / ต้องปรับปรุง
💡 แสงสว่าง: เพียงพอ / มืดเกินไป / สว่างเกินไป
🖥️ อุปกรณ์ที่เห็น: [รายการ]
📝 ข้อสังเกต: [สิ่งที่น่าสนใจ]
⭐ คะแนนรวม: [1-10]
```

### 1.5 Desk Cleanliness — ประเมินโต๊ะทำงาน

**Basic:**
```
ประเมินความสะอาดและความเป็นระเบียบของโต๊ะในภาพ ให้คะแนน 1-5 ดาว พร้อมคำอธิบาย
```

**Detailed:**
```
คุณคือผู้ตรวจสอบความสะอาด ประเมินโต๊ะนี้:
⭐ คะแนน: [1-5 ดาว]
📦 สิ่งของบนโต๊ะ: [รายการ]
✅ จุดที่ดี: [สิ่งที่เป็นระเบียบแล้ว]
⚠️ จุดที่ควรปรับปรุง: [สิ่งที่ควรจัดใหม่]
💡 คำแนะนำ: [1-3 ข้อ]
```

## 2. General Vision Prompts / Prompt ทั่วไป

### 2.1 อธิบายภาพ

```
อธิบายสิ่งที่เห็นในภาพนี้อย่างละเอียด ทั้งวัตถุ สี ตำแหน่ง และบรรยากาศโดยรวม
```

```
Describe this image in detail. Include: main subjects, colors, composition, and mood.
```

### 2.2 ถามตอบเกี่ยวกับภาพ

```
ดูภาพนี้แล้วตอบคำถามต่อไปนี้:
1. มีอะไรอยู่ในภาพบ้าง?
2. สิ่งใดดูโดดเด่นที่สุด?
3. เวลาหรือสถานที่คืออะไร?
```

### 2.3 สรุปและแนะนำ

```
วิเคราะห์ภาพนี้และให้คำแนะนำที่เป็นประโยชน์ในบริบทที่เกี่ยวข้องกับสิ่งที่เห็น
```

## 3. Advanced Prompts / Prompt ขั้นสูง

### 3.1 Multi-language Response

```
Analyze this image and provide the response in both Thai and English.
Thai section: [การวิเคราะห์ภาษาไทย]
English section: [English analysis]
```

### 3.2 Structured Data Extraction

```
Extract structured information from this image:
- List all text visible (if any)
- List all objects and their approximate count
- Describe the overall scene in one sentence
Format as JSON.
```

### 3.3 Comparison / Before-After

```
วิเคราะห์ภาพนี้และประเมินว่าสถานการณ์ที่เห็นอยู่ในระดับใด
เปรียบเทียบกับมาตรฐานที่ดี และบอกว่าต้องปรับปรุงอะไรบ้างเพื่อให้ดีขึ้น
```

### 3.4 Safety Check

```
ตรวจสอบภาพนี้เพื่อหาสิ่งที่อาจก่อให้เกิดอันตรายหรือความไม่ปลอดภัย
ถ้าพบ ให้ระบุและแนะนำวิธีแก้ไข ถ้าไม่พบ ให้บอกว่าปลอดภัยดี
```

### 3.5 Educational Context

```
คุณคือครูที่อธิบายภาพนี้ให้นักเรียนมัธยมต้นฟัง
อธิบายสิ่งที่เห็นด้วยภาษาง่ายๆ และตั้งคำถามชวนคิด 2 ข้อ
```

## Tips การปรับ Prompt

| ต้องการ | วิธีปรับ |
|---------|---------|
| คำตอบสั้นลง | เพิ่ม "ตอบสั้นๆ ไม่เกิน 3 บรรทัด" |
| คำตอบเป็นภาษาอังกฤษ | เพิ่ม "Respond in English only" |
| Format เป็น JSON | เพิ่ม "Respond in valid JSON format only" |
| AI ไม่เดา | เพิ่ม "ถ้าไม่แน่ใจ ให้บอกว่าไม่แน่ใจ อย่าเดา" |
| คำตอบละเอียดขึ้น | เพิ่ม "อธิบายอย่างละเอียด พร้อมเหตุผล" |
| เน้น accuracy | เพิ่ม "วิเคราะห์จากสิ่งที่เห็นในภาพจริงๆ เท่านั้น" |
