## Canva Tips
- Template type: **Certificate** → customize from any dark-themed template
- ใช้ "Shapes" → Rectangle ทำ border และ accent bars
- ใช้ "Text" → ตั้ง font และ color ตาม spec ด้านบน
- ใส่ QR Code ไปยัง GitHub repo ที่มุมล่างขวา (ใช้ Canva QR code element)
- Export: **PDF Print** (300 dpi) สำหรับพิมพ์จริง
- ขนาดกระดาษ: A4 Landscape (297 × 210 mm)
- พิมพ์บนกระดาษ 120–160 gsm สำหรับความสวยงาม
