README DESIGN
=============

# ESP32-CAM AI Vision Workshop

Build Intelligent Camera Systems with ESP32-CAM and Generative AI

---

## Overview

This workshop introduces students to:

* Internet of Things (IoT)
* Computer Vision
* REST API
* Generative AI
* Prompt Engineering

Students will build an AI-powered camera system capable of analyzing images using modern Vision AI models.

---

## Learning Objectives

Participants will learn how to:

* Configure ESP32-CAM
* Connect devices to Wi-Fi
* Capture images
* Send data through APIs
* Use Vision AI models
* Design effective prompts
* Build AI-powered mini projects

---
